# Villa Punto de Vista — Executive Dashboard

Sales intelligence dashboard for Villa Punto de Vista, connecting PipelineCRM → Supabase → live charts.

## Features
- Live data from Supabase (auto-refreshes)
- Period filter: This Week, Last Week, This Month, Last Month, This/Last Quarter, This/Last Year, All Time
- 5 tabs: Overview, Pipeline, Traffic & Source, Geography, Guest Satisfaction
- Click any chart to drill down into the individual deals behind it

## Live URL
https://mauriciotech.github.io/villapunto-dashboard/

## Tech Stack
- Vanilla HTML/CSS/JS
- Chart.js 4.4
- Supabase REST API
